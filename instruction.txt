Инструкция к приложению Converter версии 0.0.1.2

(1) =====
    При помощи кнопки "Выбор файла с источником данных" 
    можно выбрать файл c расширением .xlsx,
    с листами, откуда будут взяты данные.
    (КА)

(2) =====
    После выбора файла появляется меню выбора 
    необходимых листов (поиск ресурсов происходит по выбранным листам).

(3) =====
    При помощи кнопки "Выбор папки с фалами ЛСР" 
    можно выбрать папку, в которой содержатся фалы ЛСР c расширением .xlsx,
    файлы с другими расширениями и внутренние папки не будут доступны для выбора
    (только файлы в текущей директории)
    Также присутствует опция "Выбрать все".
    (ЛСР)

(4) =====
    Кнопка "Запустить обработку файла" запускает процесс обработки, 
    если он запустился, интерфейс на некоторое время станет недоступным, 
    после появится сообщение об успешном завершении обработки.

    *В правом верхнем углу доступны меню изменения маштаба приложения и перезагрузки,
    для исправления ошибок в выборе файлов или для запуска дальнейшей обработки 
    (возвращает приложение в начальное состояние).
