Инструкция к приложению Converter версии 0.0.1.0

(1) =====

Выбор файла осуществляется при помощи кнопки на панели в левой части главного окна
После выбора файла кнопка будет недоступна, для повторного выбора необходимо 
перезагрузить приложение при помощи кнопки в правом верхнем углу

(2) =====

В поле "Листы с источниками данных" необходимо выбрать те листы, 
из которых будет взята информация для сопоставления

В поле "Листы для заполнения данными" необходимо выбрать те листы,
в которые необходимо поместить искомую информацию

В поле "Лист для заполнения отсутствующими данными" можно указать тот ЛИСТ, 
в котором будет формироваться таблица исключений - позиций, которые не были найдены

(3) =====

По умолчанию обработанный файл сохраняется в той же директории,
что и взятый на обработку, с добавлением в конце названия слова "Обработанный"
